# servergo

Go program that starts a server and reponds with request
information; aditionally, visit the endpoint `/lissajous` to
see a GIF of the Lissajous Figures, optionally passing a query
parameter: cycles, to specify the number of cycles for the
parametric function
(The Go Programming Language, Alan A. A. Donovan and
Brian W. Kernighan).

## Setup

```sh
go install github.com/santiago-rodrig/servergo
```
